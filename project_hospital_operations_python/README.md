Hospital Operations Analytics (Python)

Tools: Python, Pandas, NumPy, Matplotlib
Skills: Data cleaning, EDA, data visualization

Summary:
This project analyzes patient, staff, morale, and attendance patterns for a simulated hospital system. I cleaned the data, handled missing values, explored trends, and visualized patterns using Python.

Key Steps:

Imported patients, staff, services, and scheduling data

Cleaned and validated datasets

Created visualizations for age, morale, attendance

Identified operational patterns and bottlenecks

Interpreted results in a real-world healthcare context
